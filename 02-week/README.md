# Week 2: UI Building, Navigation & State Management Basics

Welcome to Week 2 of the Flutter Bootcamp! This week builds upon your foundational knowledge by introducing essential concepts for creating interactive, multi-screen applications. You'll learn how to navigate between screens, handle user input through forms, manage application state, and enhance your apps with custom assets and themes. By the end of the week, you'll be able to create complete, polished applications with multiple screens and interactive features.

## Topics Covered

- **Navigation**: Master screen-to-screen navigation using Flutter's Navigator, including pushing and popping routes, passing data between screens, and handling navigation results.
- **Forms and User Input**: Learn to collect and validate user input using TextFields, Form widgets, controllers, and various input validation techniques.
- **Basic State Management**: Understand how to manage and update application state using setState, and learn when to lift state up to share data between widgets.
- **Assets and Theming**: Enhance your apps by adding images, implementing custom fonts, and creating light and dark themes for a professional look and feel.

## Learning Objectives

By the end of Week 2, you will be able to:
- Navigate between multiple screens and pass data effectively
- Create forms with proper validation and error handling
- Manage application state using setState and understand state lifting patterns
- Add custom assets (images and fonts) to your applications
- Implement light and dark themes for better user experience
- Build a complete multi-page application with forms and navigation
